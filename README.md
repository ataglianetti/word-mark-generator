# Word Mark Generator

Interactive word mark / logotype generator with customizable text effects, font selection, color palettes, and high-res transparent PNG export. Built for rapid iteration on brand wordmarks.

Originally pulled from a Figma Make file for local development.

## Features

- 9 Google Fonts with adjustable size, spacing, and kerning
- Stackable text effects: flip, blur, stroke, shadow, glow, emboss
- Per-letter effect targeting
- Color palettes with saturation control
- Bokeh circle decorative background
- 4x resolution transparent PNG export
- URL state persistence for shareable configurations
- Collapsible controls panel

## Setup

```bash
npm install
```

## Development

```bash
npx vite --port 5174
```

## Export

Click the download button in the controls panel to export a transparent PNG at 4x resolution, suitable for use in Adobe Illustrator or other vector tools.
